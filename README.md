# spock-rock-game

Similar to rock paper scissor game

Check the demo hosted on https://filipeqs.github.io/spock-rock-game/

---

## Instructions

First clone this repository

```bash
$ git clone https://github.com/filipeqs/spock-rock-game.git
```

Open index.html

```bash
open index.html
```

## Features

- Play Game
- Record score
- Reset score
